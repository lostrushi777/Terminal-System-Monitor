# Terminal-System-Monitor

![altimage](https://github.com/lostrushi777/Terminal-Dashboard/blob/main/monitor_preview.png?raw=true)

# Features
- Real-time CPU usage monitoring
- CPU temperature monitoring
- RAM usage statistics
- Disk usage information
- Network upload and download activity
- System uptime tracking
- Battery status (when available)
- Hostname, operating system, and kernel information
- Color-coded CPU usage status

# Requirements
```bash
Python 3.8+
```

```bash
psutil
```

# INSTALLATION!
HTTPS
```bash
git clone https://github.com/lostrushi777/Terminal-System-Monitor.git
```
SSH

git clone git@github.com:lostrushi777/Terminal-System-Monitor.git

# Usage

Run the dashboard:
```bash
python monitor.py
```
```bash
python3 monitor.py
```
Stop the dashboard with:

Ctrl + C
